Este proyecto consiste en el desarrollo de un sitio web e-commerce realizado por un equipo de estudiantes, login, responsive design y almacenamiento en LocalStorage. El objetivo es construir una tienda funcional donde un usuario administrador pueda gestionar productos y los visitantes puedan navegar, filtrar y ver detalles de los mismos.
